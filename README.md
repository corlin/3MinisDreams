# 晨梦日记 (Morning Dream Diary)

一个专注于正向激励的移动应用，帮助用户记录未来一周的愿望目标，通过定期回顾实现情况来建立持续的成就感和动力。

## 核心理念

每天早晨花3分钟时间，记录一周后希望实现的小愿望。一周后回顾这些愿望的实现情况，庆祝每一个达成的目标，从而建立正向反馈循环，激励持续的个人成长。

## 功能特性

- ✨ 记录一周后的愿望和目标
- ⏰ 3分钟专注计时，提升记录质量
- 🎯 愿望分类管理（个人成长、事业、健康等）
- 🏆 成就回顾系统，庆祝每一个实现的目标
- 📈 成功率统计和激励性反馈
- 🔔 个性化提醒和激励消息
- 🌟 简洁优雅的界面设计

## 技术栈

- React Native (Expo)
- TypeScript
- React Navigation
- 本地存储

## 开发环境设置

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 在模拟器或设备上运行：
```bash
npm run ios     # iOS
npm run android # Android
npm run web     # Web
```

## 使用方法

1. **记录愿望** - 每天早晨花3分钟记录一周后想要实现的愿望
2. **专注时间** - 利用内置计时器保持专注，提升记录质量
3. **分类管理** - 将愿望按个人成长、事业、健康等分类整理
4. **定期回顾** - 一周后系统自动提醒回顾愿望实现情况
5. **庆祝成就** - 标记实现的目标，获得成就感和持续激励
