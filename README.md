# 晨梦日记 (Morning Dream Diary)

一个用于记录和分析梦境的移动应用，帮助用户追踪梦境模式并获得个人洞察。

## 功能特性

- 📝 记录梦境内容和情绪
- 🏷️ 梦境标签和分类
- 📊 梦境统计和分析
- 🔍 梦境搜索功能
- 🌙 简洁优雅的界面设计

## 技术栈

- React Native (Expo)
- TypeScript
- React Navigation
- 本地存储

## 开发环境设置

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm start
```

3. 在模拟器或设备上运行：
```bash
npm run ios     # iOS
npm run android # Android
npm run web     # Web
```

## 项目结构

```
src/
├── components/     # 可复用组件
├── screens/        # 页面组件
├── navigation/     # 导航配置
├── services/       # 业务服务
├── types/          # TypeScript 类型定义
├── utils/          # 工具函数
└── constants/      # 常量定义
```
